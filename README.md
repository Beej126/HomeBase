# Mission Control

A PowerShell-based WPF application that hosts multiple resizable and draggable WebView2 controls in a single window.

## Features (Planned)

- Multiple WebView2 browser instances
- Resizable and draggable web views
- Modern dark-themed UI
- Tab management for different views
- Custom layout configurations

## Requirements

- Windows 10/11
- PowerShell 5.1 or higher
- .NET Framework 4.7.2 or higher
- Microsoft Edge WebView2 Runtime

## Installation

1. Clone this repository
2. Ensure WebView2 Runtime is installed (usually comes with Windows)
3. Run `MissionControl.ps1`

## Usage

```powershell
.\MissionControl.ps1
```
